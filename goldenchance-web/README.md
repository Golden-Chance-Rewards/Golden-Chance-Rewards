# Golden Chance Rewards Web App

A Next.js application for the Golden Chance Rewards system.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` - Next.js app router pages
- `src/components/` - Reusable UI components
- `src/state/` - State management
- `src/api/` - API utilities
- `src/hooks/` - Custom React hooks
- `src/utils/` - Utility functions
- `src/styles/` - Styling files
- `src/config/` - Configuration files

## Troubleshooting

- Ensure Node.js version 20 or higher is installed.
- If you encounter issues with dependencies, try deleting `node_modules` and running `npm install` again.
